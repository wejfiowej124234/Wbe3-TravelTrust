# 仓库提交范围：该上传 / 不该上传

## 应上传（提交到 GitHub）

| 类型 | 路径/说明 |
|------|------------|
| 代码 | `crates/`（core、api、web）、`contracts/`（含 README） |
| 文档 | `docs/` 下全部（00～08-5、多维度报告等） |
| 运维 | `ops/RUNBOOK.md` |
| 证据说明与模板 | `evidence/README.md`、`evidence/GO_YYYYMMDD_template/`、`evidence/GO_placeholder/` |
| 脚本 | `scripts/` 下全部 |
| CI/PR | `.github/`（PR 模板、workflows） |
| 根配置 | `README.md`、`Cargo.toml`、`.gitignore`、`.env.example`（若有） |

## 不应上传（已由 .gitignore 排除）

| 类型 | 路径/说明 |
|------|------------|
| 构建产物 | `target/`、`dist/` |
| 环境与密钥 | `.env`、`.env.*`、`*.pem`、`*.key`、含 `credentials` 的文件 |
| 真实 evidence bundle | `evidence/GO_YYYYMMDD/`（8 位日期目录，过门时产出，可能含内部信息） |
| 备份/临时 | `*.rs.bk`、`*.bak`、`*.tmp` |
| 锁文件（可选） | `Cargo.lock` 已忽略；若希望可复现构建可改为提交 |

## 提交前自检

- [ ] 未包含 `.env` 或真实密钥
- [ ] 未包含真实姓名/邮箱/内部链接（除占位说明外）
- [ ] `evidence/` 下仅 README、模板、placeholder，无真实 GO_ 日期目录
